# Udemy Courses Exploratory Data Analysis (EDA)

## Abstract
This project presents a structured Exploratory Data Analysis (EDA) of a Udemy online courses dataset. The analysis investigates patterns in course pricing, subject distribution, subscriber engagement, review activity, and content structure. The goal is to extract insights that help explain trends in course performance using statistical and visualization methods.

---

## Introduction
Online learning platforms generate large amounts of data that can be used to understand learner behavior and course design. This project analyzes a Udemy courses dataset to explore relationships between pricing models, subject categories, subscriber counts, reviews, and instructional content features.

---

## Dataset
The dataset includes information such as:

- Course title and URL  
- Subject category  
- Paid/free indicator and price  
- Number of subscribers  
- Number of reviews  
- Number of lectures  
- Content duration  
- Published timestamp  

---

## Methodology

### Data Cleaning
- Checked and removed duplicate records  
- Verified and handled missing values  
- Standardized column names  
- Ensured correct data types (numerical, categorical, timestamps)

### Exploratory Analysis
This project explores:
- Course distribution across subjects
- Free vs paid course proportions
- Subscriber and review patterns across categories
- Relationship between reviews and subscribers
- Differences in duration and lecture counts by subject
- Common keywords in course titles (word cloud)

### Visualization
Used:
- Bar charts
- Histograms
- Scatter plots
- Correlation-focused visuals
- Word cloud for course title text

---

## Key Findings (Summary)
- Course availability differs strongly by subject category.
- Subscribers and reviews typically show a strong positive relationship.
- Paid and free courses show different engagement patterns.
- Course duration and lecture counts vary across subject areas.
- Certain keywords appear frequently among popular course titles.
